# Unit2 GitHub Pages setup

Upload these into the `cjohnson155/aeronautical_propulsion` repo.

## Files

- `index.html`
  - Replaces the current top-level `index.html`.
  - Adds a fixed `Unit 2` link.
  - Changes `/vite.svg` and `/main.jsx` to relative paths: `./vite.svg` and `./main.jsx`.

- `Unit2/index.html`
  - New Unit 2 homepage.
  - It will render at:
    `https://cjohnson155.github.io/aeronautical_propulsion/Unit2/`

## GitHub steps

1. In the repo root, replace the existing `index.html` with the provided `index.html`.
2. Create a folder called `Unit2`.
3. Inside `Unit2`, create `index.html` and paste the provided `Unit2/index.html`.
4. Commit changes.
5. Open:
   `https://cjohnson155.github.io/aeronautical_propulsion/Unit2/`

Do not change Settings → Pages unless the current site is not already publishing.
